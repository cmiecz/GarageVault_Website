What is Store Management?

Store management lets you track where you purchased items, keep purchase history, and quickly reorder items from your favorite retailers. Garage Vault comes with popular stores pre-loaded, and you can add your own custom stores.

Adding a Custom Store

Want to add your local hardware store, specialty shop, or online retailer?

Step-by-Step:

  1. Go to Settings ‚Üí Store Customization
  2. Tap "Add Store" or the "+" button
  3. Fill in the store details:
    • Store Name (e.g., "Joe's Hardware", "Harbor Freight")
    • Website (optional - for quick reordering)
    • Phone Number (optional)
    • Address (optional)
  4. Tap "Save"

Your custom store is now available when adding purchase information to items!

Adding Store Logos

Store logos help you quickly identify where items were purchased.

Option 1: Auto-Fetch Logo (Easiest)

The app automatically fetches logos as you type!

  1. When creating a new store, start typing the Store Name
  2. As you type, the app automatically searches for the store's logo online
  3. If found, the logo appears instantly below the name field
  4. Continue filling in other details (website, phone, address)
  5. Tap "Save"

Note: This works best for well-known retailers like Harbor Freight, Ace Hardware, Menards, etc.

Pro Tip: Type the full official name (e.g., "Harbor Freight Tools" not just "Harbor") for best results.

Option 2: Upload Your Own Logo

  1. When creating or editing a store
  2. Tap "Add Logo" or the camera icon
  3. Choose an option:
    • Take Photo - Photograph the store sign or logo
    • Choose from Gallery - Select an existing photo
  4. Crop or adjust if needed
  5. Tap "Save"

Pro Tip: Take a photo of the storefront for local shops - it helps you remember which store is which!

Editing Store Information

Need to update a store's details?

  1. Go to Settings ‚Üí Store Customization
  2. Find the store you want to edit
  3. Tap on the store name
  4. Update any information:
    • Name
    • Logo
    • Website
    • Contact information
  5. Tap "Save"

Deleting a Custom Store

  1. Go to Settings ‚Üí Store Customization
  2. Find the store you want to remove
  3. Swipe left on the store name (or tap the "..." menu)
  4. Tap "Delete"
  5. Confirm deletion

Note: You cannot delete the built-in stores (Home Depot, Lowes, Amazon), but you can choose not to use them.

Warning: Deleting a store won't delete items purchased from that store, but the purchase history will show "Unknown Store" instead.

Using Stores When Adding Items

When you add or edit an item, you can track where it was purchased:

Adding Purchase Information:

  1. Open an item or create a new one
  2. Scroll to the Purchase Information section
  3. Tap "Select Store"
  4. Choose from:
    • Built-in stores (Home Depot, Lowes, Amazon)
    • Your custom stores
  5. Fill in additional details (optional):
    • Purchase Date
    • Purchase Price
    • Receipt Photo
  6. Tap "Save"

The store logo will now appear on the item's detail page!

Quick Reorder Links

Home Depot, Lowes & Amazon

For items purchased from built-in stores, you get quick reorder buttons:

  1. Open an item purchased from Home Depot, Lowes, or Amazon
  2. Look for the "Buy Again" or store logo button
  3. Tap the button
  4. Your device's browser opens to that store's website
  5. Search for the item to reorder

Custom Stores with Websites

If you added a website URL to your custom store:

  1. Open an item from that store
  2. Tap the store logo or "Visit Store" button
  3. Your browser opens to the store's website

Viewing Items by Store

Want to see everything you've bought from a specific store?

  1. Go to Inventory or Dashboard
  2. Tap the Filter icon
  3. Select "Filter by Store"
  4. Choose the store
  5. Only items from that store will display

Great for:

  • Seeing what you typically buy from each store
  • Planning shopping trips
  • Tracking spending by retailer

Best Practices

Store Names

‚úÖ DO:

  • Use clear, recognizable names ("Harbor Freight Tools")
  • Include location if you have multiple ("Ace Hardware - Downtown")
  • Be consistent with naming

‚ùå DON'T:

  • Use vague names ("Store 1", "Hardware")
  • Create duplicates of the same store
  • Use abbreviations others won't understand

Store Logos

‚úÖ DO:

  • Use clear, recognizable logos
  • Take photos in good lighting
  • Crop logos tightly for best display
  • Use auto-fetch when possible

‚ùå DON'T:

  • Use blurry or dark photos
  • Include too much background
  • Skip logos entirely - they help identify stores quickly!

When to Add Custom Stores

Add a custom store if you:

  • Shop there frequently (3+ times per year)
  • Buy specialty items only available there
  • Want to track spending at that location
  • Need to remember the store for warranty purposes

Skip custom stores if you:

  • Only shopped there once
  • Can't remember the store name
  • Already have a similar store in your list

Common Questions

Q: Can I have more than one store with the same name?

A: Yes, but add location details to tell them apart (e.g., "Ace Hardware - Main St" vs "Ace Hardware - Oak Ave")

Q: What if I can't find the store logo online?

A: Take a photo of the storefront, business card, or shopping bag with the logo

Q: Do I need to add stores I've only used once?

A: No, only add stores you'll shop at again. For one-time purchases, you can leave the store blank or use "Other"

Q: Can I edit the built-in stores (Home Depot, Lowes, Amazon)?

A: No, but you don't need to - they're already configured with logos and purchase links

Q: Will my custom stores sync to my household members?

A: Yes! All household members see the same custom stores

Q: What happens to my items if I delete a store?

A: Items remain unchanged, but the store name will show as "Unknown" in purchase history
