Understanding Categories

What are Categories?

Categories help you organize your items by type, making it easy to find what you need quickly. Think of categories as the "what" of your inventory organization.

Built-in Categories

Garage Vault comes with several pre-configured categories:

  • Tools - Power tools, hand tools, and accessories
  • Fasteners - Screws, nails, bolts, nuts, and washers
  • Paint - Paint cans, stains, primers, and finishes
  • Automotive Fluids - Motor oil, coolant, brake fluid, and more
  • Equipment - Lawn mowers, snow blowers, chainsaws, etc.

Custom Categories

You can create custom categories for your specific needs, such as:

  • Electrical supplies
  • Plumbing parts
  • Holiday decorations
  • Sports equipment
  • Seasonal items

Managing Categories

Creating a Custom Category

  1. Navigate to Dashboard
  2. Scroll to the Categories section
  3. Tap the "+" or "Add Category" button
  4. Enter your category name
  5. Choose an icon (optional)
  6. Tap "Save"

Reordering Categories

  1. Go to Dashboard
  2. Find the Categories section
  3. Tap and hold on a category
  4. Drag it to your desired position
  5. Release to drop in place

Your category order will be saved automatically and applied across the app.

Filtering by Category

From the Dashboard:

  1. Tap any category card to instantly filter your inventory
  2. Only items in that category will be displayed
  3. Tap "All Items" or "Clear Filter" to see everything again

From the Inventory Screen:

  1. Tap the filter icon (funnel)
  2. Select one or more categories
  3. Apply filter to narrow down results

Hiding/Showing Categories on Dashboard

  1. Go to Settings ‚Üí Dashboard Preferences
  2. Toggle categories on/off to show or hide them
  3. Only selected categories will appear on your dashboard

Understanding Storage Locations

What are Storage Locations?

Storage Locations are the physical places where you keep your items. Think of them as the "where" of your inventory organization. They help you track exactly which drawer, bin, shelf, or toolbox contains each item.

Storage Types

Garage Vault supports various storage types:

  • Packout - Milwaukee Packout modular storage systems
  • Bin - Storage bins and containers
  • Drawer - Individual drawers in toolboxes or cabinets
  • Shelf - Shelving units and racks
  • Cabinet - Wall or floor cabinets
  • Toolbox - Portable toolboxes and chests
  • Other - Any custom storage solution

Setting Up Storage Locations

üí° Premium Feature

Creating and editing storage locations requires Premium. Learn more about Premium.

Creating a Storage Location (Premium)

Creating and editing storage locations is a Premium feature:

  1. Go to Inventory ‚Üí Storage Locations
  2. Tap the "+ Add Storage" button
  3. Fill in the details:
    • Name - Give it a clear, descriptive name (e.g., "Red Toolbox", "Shelf 2")
    • Type - Select the storage type
    • Color - Choose a color for visual organization (optional)
    • Description - Add any notes about the storage location (optional)
  4. Tap "Save"

Adding Storage Positions

Storage positions help you track the exact spot within a storage location:

  1. Open an existing storage location
  2. Tap "Add Position" or similar option
  3. Enter position details (e.g., "Slot 3", "Top Drawer", "Back Left Corner")
  4. Save

Color Coding Your Storage

Color coding makes it easy to visually identify storage locations:

  1. When creating or editing a storage location
  2. Tap the color selector
  3. Choose a color that matches your physical storage (if colored) or pick a color for easy identification
  4. The color will appear next to the storage name throughout the app

Viewing Items in a Storage Location (Premium)

Once you've created storage locations, you can view them and their contents:

  1. Go to Inventory ‚Üí Storage Locations
  2. Tap on any storage location
  3. You'll see:
    • All items stored in that location
    • Item count (automatically tracked)
    • Storage details (type, color, description)

Sorting and Filtering Storage

Sort Storage By:

  • Name (A-Z or Z-A)
  • Type (group similar storage together)
  • Item count (most/least items)
  • Date created (newest/oldest)

Filter Storage By:

  • Storage type (show only Bins, Drawers, etc.)
  • Items with low stock
  • Empty storage locations

To sort or filter:

  1. Go to Storage Locations
  2. Tap the sort/filter icon
  3. Select your preferred option

Using QR Code Labels

üí° Premium Feature

Creating storage locations and generating QR codes is a Premium feature. Learn more about Premium.

What are QR Code Labels?

QR code labels are physical stickers or tags you can attach to your storage locations. Scanning them instantly shows you what's inside without opening drawers or bins.

Generating QR Codes (Premium)

  1. Open a storage location
  2. Tap "Generate QR Code" or the QR icon
  3. The app will create a unique QR code for that location
  4. Tap "Print" or "Save" to export the QR code
  5. Print the QR code and attach it to your physical storage

Pro Tip: Use weatherproof labels for outdoor storage or areas exposed to moisture.

Scanning QR Codes

  1. Open the Garage Vault app
  2. Tap the camera/scan icon (usually on the main screen)
  3. Point your camera at the QR code on your storage
  4. The app will automatically:
    • Open that storage location
    • Show all items inside
    • Display item quantities and details

Benefits of QR Codes

  • Instant Access - No searching through menus
  • Faster Inventory Checks - Scan to see what's inside
  • Easy for Family - Others can find items without asking
  • Perfect for High Shelves - Check contents without climbing
  • Workshop Efficiency - Quick identification in busy spaces

Best Practices

Category Best Practices

  1. Keep it Simple - Don't create too many categories; 10-15 is usually enough
  2. Be Consistent - Always use the same category for similar items
  3. Use Broad Categories - Instead of "Impact Drivers" and "Drills," use "Power Tools"
  4. Review Regularly - Merge or delete unused categories every few months

Storage Location Best Practices

  1. Label Everything - Name storage locations clearly (e.g., "Garage Wall Cabinet #2" not "Cabinet")
  2. Use QR Codes - Print and attach QR codes to all major storage
  3. Color Code Logically - Match colors to physical storage or use a consistent system
  4. Track Positions - For large storage units, add position details (drawer, slot, shelf level)
  5. Consolidate When Possible - Don't create separate locations for every small container
  6. Photo Your Storage - Add a photo to each storage location for quick visual reference
  7. Group by Zone - Organize storage by areas (Workshop, Garage, Shed, Basement)

Combining Categories and Storage

The most powerful organization happens when you combine both:

Example 1: Workshop Organization

  • Category: Fasteners
  • Storage: Red Packout Bin (Position: Slot 2)
  • Result: All your screws are categorized as Fasteners AND you know they're in the Red Packout, Slot 2

Example 2: Seasonal Equipment

  • Category: Equipment
  • Storage: Shed Shelf 3 (Position: Back Left)
  • Result: Your snow blower is categorized as Equipment AND stored on Shed Shelf 3

Example 3: Paint Organization

  • Category: Paint
  • Storage: Blue Cabinet (Position: Top Shelf)
  • Result: All paints are in one category AND one physical location for easy access
